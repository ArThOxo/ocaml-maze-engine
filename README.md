# 🕹️ OCaml Maze Generator & Solver

![OCaml](https://img.shields.io/badge/OCaml-4.14-orange.svg)
![Build](https://img.shields.io/badge/build-dune-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A high-performance algorithmic project written entirely in **OCaml**. It generates perfect random mazes using depth-first search (DFS) and solves them by finding the path from Start (`S`) to Exit (`E`), leaving a trail of breadcrumbs (`.`).
